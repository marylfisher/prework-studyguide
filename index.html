HTML

  • HTML stands for Hypertext Markup Language
  • HTML contains the basic framework for all web pages.
  • Elements include: Text, Image, Header, List, Meta, Content division, and Semantic.
  • The head element contains information about the web page
  • The head element contains metadata that helps the browser render the page correctly.
  • Head elements are not visable to the user.
  • The body element represents the visual content shown to the user
  • Begin by declaring doctype (not case sensitive) then openign HTML tag
  • Input attributes. Attributes define additional information about an element, i.e. language.

CSS

  • CSS stands for Cascading Style Sheets
  • CSS gives the content style
  • Degins colors, aesthetics, visual layout and organization
  • Inline CSS styles an element directly in an HTML file
  • Internal style sheet contains CSS rules for the head section of the HTML file
  • External CSS style sheet (most popular) keeps all CSS rules in a seperate file
  • A CSS rule has a selctor than a declaration box that consists of a propert and value
  • A margin indicates how much space we want outside an element
  • A padding we want around the contents inside an element
  • You can apply CSS to the whoel page or a class atrribute using a class selector

Git and Github

  • Git allows you to manage versions of your project
  • Github is a cloud hosting service that allows us to keep track of our projects inside repositories that are stored remotely.
  • git status: checks what branch we are currently on
  • git check out -b branch-name: creates a new branch and switches to it
