HTML

  • HTML stands for Hypertext Markup Language
  • HTML contains the basic framework for all web pages.
  • Elements include: Text, Image, Header, List, Meta, Content division, and Semantic.
  • The head element contains information about the web page
  • The head element contains metadata that helps the browser render the page correctly.
  • Head elements are not visable to the user.
  • The body element represents the visual content shown to the user
  • Begin by declaring doctype (not case sensitive) then openign HTML tag
  • Input attributes. Attributes define additional information about an element, i.e. language.

CSS

  • CSS stands for Cascading Style Sheets
  • CSS gives the content style
  • Degins colors, aesthetics, visual layout and organization
  • Inline CSS styles an element directly in an HTML file
  • Internal style sheet contains CSS rules for the head section of the HTML file
  • External CSS style sheet (most popular) keeps all CSS rules in a seperate file
  • A CSS rule has a selctor than a declaration box that consists of a propert and value
  • A margin indicates how much space we want outside an element
  • A padding we want around the contents inside an element
  • You can apply CSS to the whole page or a class atrribute using a class selector
  • CSS is linked within head

Git

  • Git allows you to manage versions of your project
  • Github is a cloud hosting service that allows us to keep track of our projects inside repositories that are stored remotely.
  • git status: checks what branch we are currently on
  • git check out -b branch-name: creates a new branch and switches to it

JavaScript

  • JaveScript (JS) handles logic and allows users to interact with a web site
  • JS makes web pages interactive
  • Nasic JS features include variables, strings and data types
  • Browser Application Prgramming Interfaces (APIs) are additional tools on top of JS
  • APIs are built into the web browser, allow for maniupating video stream, 3D grphics, more dynamic CSS or HTML
  • Variables are containers that store values, declare variable with const or let and semi colon ends statement
  • Control flow is the order in which a computer executes code in a script
  • JS is case sensitive
  • An operator is a mathematical symbol that produces a result based on two valuables
  • Conditionals are code structures used to test if an expression returns true or not
  • Functions package functionality you wish to reuse
  • Real interactivity requires event handlers
  • JS is linked within body
  • An array is a single variable that holds a group of data
  • Index allows us to access individual items in an array
  • A for loop is a predictable pattern of indices that performs a task on all the items in an array by allowing a single code block to be executed over and over again
  • For loop has three statements: the starting point loop, then condition (how long it will run), last allows array to iterate
  • A function is a set of instructions that tells a computer how to perform a certain task
  • Must define function, can use function declaration which uses function keyword and code declaration
  • We must call a function in the code at the spot we want it to execute
